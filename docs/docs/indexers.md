# Indexers


## How to add a new chain, e.g. Acala

### Subsquid 
create an indexer

### subledgr
- activate the asset in DB & vue3

### REST api

- add the RPC endpoint to the https://api.metaspan.io/api/${assetId}/

### Exchange Rates

- add the asset to the exchange rate workers

### Firewall

- add /api/acala to the proxy pass
